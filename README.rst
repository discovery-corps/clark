=====
Clark
=====

A high performance bittorrent tracker frontend, written in Python.

Supports Python 2.7, Python 3.5, and PyPy.

To run lint and unit tests, run::

    tox

To run the app and play around, run::

    tox -evenv gunicorn clark.app:api
